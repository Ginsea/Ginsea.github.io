1. R语言安装

  • 从CRAN 中下载R语言安装程序，该程序可以在Windows，Linux以及Mac平台中使用。
  • 从RStudio中下载R语言编译器RStudio，该程序可以在Windows, Linux以及Mac平台中使用。

2. R语言使用

  • R语言区分大小写，help(),Help(),HELP()为不同的函数，但只有第一个是正确的；
  • R使用 <- 作为赋值符号，但也允许使用=为对象赋值，但这么写的R程序不多；

  • R语言中将反斜杠（\ ）作为一个转义符，因此即使在Windows中使用路径，也需要使用正斜杠（/）

  • 使用demo()函数了解R语言能够作出何种图形

  • 通过help()函数获取帮助信息
